## 3. Define the libs and set how to use them ‚ùé

### Graphics Library

- Vulkan
  high-performance, modern GPU API
- Vulkan-Hpp
  Modern C++ Vulkan bindings.
- DEARImGUI
  fast debug/dev UI (optional, good for tools/debug overlays)
- GLFW
  cross-platform window and input handling
- GLM
  OpenGL-style math library: matrices, vectors.
  Use GLM to build model, view, projection matrices.
- glslangValidator
  Compile GLSL to SPIR-V(Shaders) offline or runtime.
- Assimp
  Mesh Loading
- stb_image
  Help load PNG/JPG/TGA, etc. image files.
- Vulkan Memory Allocator (VMA)
  Eases Vulkan memory management. __if needed__
